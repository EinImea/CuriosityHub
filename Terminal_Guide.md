# Terminal & Git Essentials for EinImea
## 1. Navigation (The Basics)
* `pwd`: Show where I am (Print Working Directory).
* `ls -la`: Show all files, including hidden ones.
* `cd ..`: Move up one folder.
## 2. Homebrew (The Librarian)
* `brew leaves`: List only the tools I installed.
* `brew update && brew upgrade`: Keep the workshop clean.
## 3. The Git 'Heartbeat'
1. `git add .`: Gather all my changes.
2. `git commit -m "note"`: Save the snapshot.
3. `git push`: Send to GitHub.
